Entry 1: Starting My Coding Journey

Today marks the beginning of my journey into programming! I’ve always been curious about how websites and apps work, so I decided to dive in and start learning. I began with HTML and CSS, creating simple webpages with basic structure and styling. The learning curve is steep, but it's rewarding to see my progress every day!

One of the first things I learned was how to structure a webpage using HTML. Elements like headings, paragraphs, lists, and links make up the foundation of any webpage. Next, I tried out CSS to add colors, fonts, and layout adjustments. It’s amazing how a little CSS can transform a plain page into something that looks more polished and professional.

Entry 2: First Steps into HTML/CSS - Creating My Favorite Recipe Table

This week, I explored HTML tables and CSS styling to create a recipe table for my favorite dishes. Learning how to use tables to organize information was exciting because it added a new level of structure to my pages. I made a table with columns for ingredients and amounts, which made it easy to see everything in one place.

Adding CSS to style the table was a fun part of the process. I experimented with background colors, borders, and padding to make the table more readable and visually appealing. I’m proud of the result and can’t wait to use tables and styling in more projects!
